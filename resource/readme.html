JPA 2.0 Readme

JSR 317 (license) is packaged in an OSGi format for distribution with the EclipseLink Project which is the reference implimentation of the JPA 2.0 specification.

The source code is available within the project's javax.persistence Git repository. The binaries are distributed in the OSGi enabled javax.persistence_2.1.0.*.jar bundle.
